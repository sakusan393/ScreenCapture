# ScreenCapture v1.0.0 - リリースノート

## 配布ファイル

**`ScreenCapture.exe`** (約155MB)

## 動作環境

- **OS**: Windows 10/11 (64bit)
- **必要なランタイム**: なし（すべて含まれています）
- **インストール**: 不要（EXEをダブルクリックで起動）

## 使い方

### 基本操作

1. **`ScreenCapture.exe`をダブルクリック**して起動
2. **マウスで範囲を選択**してキャプチャ
3. **Esc**キーで終了

### 機能一覧

#### キャプチャ
- **画面選択**: マウスドラッグで範囲を選択
- **Ctrl+C**: キャプチャ全体をクリップボードにコピー

#### テキスト追加
- **右クリック** → 「テキスト追加」
- **ダブルクリック**で再編集
- **色の変更**: カラーピッカーで文字色を選択
- **背景色**: カラーピッカーで背景色を選択（透明に戻すボタンあり）
- **サイズ変更**: [+] [-] ボタンまたは押しっぱなしで連続変更
- **削除**: [X] ボタン
- 設定は次回起動時も保持

#### 画像追加
- **Ctrl+V**: クリップボードから画像を貼り付け
- **ドラッグ**: 移動
- **リサイズ**: 四隅のハンドルをドラッグ
- **回転**: 上部中央のハンドルをドラッグ
- **枠線**: 表示ON/OFF（10px/20pxで切り替え）
- **枠線色**: カラーピッカーで変更
- **削除**: [X] ボタン

#### ペイント機能
- **Alt**キー: ペイントモード ON/OFF
- **色の選択**: カラーピッカーで選択
- **太さの選択**: 1, 3, 5, 10 の4段階
- **Shift + ドラッグ**: 水平・垂直の直線描画
- **Ctrl+Z**: アンドゥ（元に戻す）
- **Ctrl+Y**: リドゥ（やり直す）
- **履歴回数**: 10/20/50/100から選択可能
- 設定は次回起動時も保持

#### その他
- **マウスホバー**: 赤枠が濃くなり、×ボタンが表示される
- **×ボタン**: ウィンドウを閉じる
- **Esc**キー: ウィンドウを閉じる

## ショートカットキー一覧

| キー | 機能 |
|------|------|
| **Ctrl+C** | クリップボードにコピー |
| **Ctrl+V** | 画像を貼り付け |
| **Alt** | ペイントモード ON/OFF |
| **Ctrl+Z** | アンドゥ（元に戻す） |
| **Ctrl+Y** | リドゥ（やり直す） |
| **Shift + ドラッグ** | 水平・垂直線 |
| **Esc** | 終了 |

## 使用例

### 1. 画面の一部をキャプチャして注釈を付ける
```
1. ScreenCapture.exe を起動
2. キャプチャしたい範囲をドラッグで選択
3. 右クリック → 「テキスト追加」で説明文を追加
4. Alt でペイントモードに切り替えて矢印を描く
5. Ctrl+C でクリップボードにコピー
6. Word/PowerPointに貼り付け
```

### 2. スクリーンショットに図形を描く
```
1. 範囲選択してキャプチャ
2. Alt キーでペイントモード ON
3. Shift + ドラッグで四角形の枠線を描く
4. 色や太さを変更して強調
5. Ctrl+C でコピー
```

### 3. 画像を合成する
```
1. 範囲選択してキャプチャ
2. Ctrl+V で別の画像を貼り付け
3. リサイズ・回転で配置
4. テキストで説明追加
5. Ctrl+C でコピー
```

## トラブルシューティング

### Q: 起動しない
A: Windows Defenderに除外設定が必要な場合があります。

### Q: ファイルサイズが大きい
A: .NET Runtimeを含むため、単体EXEとして配布するためです。

### Q: 更新方法は？
A: 新しい`ScreenCapture.exe`をダウンロードして置き換えてください。
