# ScreenCapture v1.0.0 ??

画面キャプチャ＋注釈ツール - 初回リリース

## ?? ダウンロード

**`ScreenCapture.exe`** (約155MB) をダウンロードして、ダブルクリックで起動できます。

## ? 機能

### ?? 画面キャプチャ
- マウスドラッグで範囲を選択
- 複数モニター対応
- Ctrl+Cでクリップボードにコピー

### ?? テキスト注釈
- 右クリックでテキスト追加
- **8色から選択可能**（白、黒、赤、黄、緑、青、水色、マゼンタ）
- **サイズ変更**: [+][-]ボタンまたは押しっぱなし
- ダブルクリックで再編集
- ドラッグで移動

### ??? 画像の貼り付け
- **Ctrl+V**でクリップボードから画像を追加
- リサイズ：四隅のハンドルをドラッグ
- 回転：上部中央のハンドルをドラッグ
- ドラッグで移動

### ?? ペイント機能
- **Ctrl**キーでペイントモード ON/OFF
- **6色から選択**（白、黒、赤、黄、緑、青）
- **4段階の太さ**（1, 3, 5, 10）
- **Shift+ドラッグ**で水平・垂直の直線
- **Ctrl+Z**でアンドゥ（元に戻す）
- **Ctrl+Y**でリドゥ（やり直す）
- **履歴回数設定**：10/20/50/100回から選択

### ?? その他
- マウスホバーで枠線表示・×ボタン表示
- 複数の要素を選択で最前面に移動
- Escキーで終了

## ?? ショートカットキー

| キー | 機能 |
|------|------|
| **Ctrl+C** | クリップボードにコピー |
| **Ctrl+V** | 画像を貼り付け |
| **Ctrl** | ペイントモード ON/OFF |
| **Shift+ドラッグ** | 水平・垂直線 |
| **Ctrl+Z** | アンドゥ |

- **OS**: Windows 10/11 (64bit)

## ?? 使用例

### 1. スクリーンショットに矢印と説明を追加
```
1. 範囲を選択してキャプチャ
2. Ctrl でペイントモード ON
3. Shift+ドラッグで矢印を描く
4. 右クリックでテキスト追加
5. Ctrl+C でコピー
6. Word/PowerPointに貼り付け
```

### 2. 画像を合成してコラージュ作成
```
1. 範囲を選択してキャプチャ
2. Ctrl+V で別の画像を貼り付け
3. リサイズ・回転で配置
4. テキストで説明追加
5. Ctrl+C でコピー
```

### 3. きれいな図形を描く
```
1. 範囲を選択してキャプチャ
2. Ctrl でペイントモード ON
3. Shift+ドラッグで四角形の枠線
4. 色や太さを変更
5. Ctrl+C でコピー
```

## ?? トラブルシューティング

### 起動しない場合
Windows Defenderに除外設定が必要な場合があります。

### Windows Defenderの警告が出る場合
1. 「詳細情報」をクリック
2. 「実行」をクリック

### ファイルサイズが大きい理由
.NET Runtimeを含むため、単体EXEとして配布しています。
